bits-tools Docs¶

Overview¶

bits-tools is a Python module that lets you work with individual bits. It has two main classes: TwoBits and RandomBits.

TwoBits¶

TwoBits stores two bits and lets you use logic gates on them.

__init__ (Constructor)¶

Arguments:

  • bit1: integer, either 0 or 1

  • bit2: integer, either 0 or 1
